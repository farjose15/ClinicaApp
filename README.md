# ClinicaApp 🏥
Sistema de Gestión Clínica - ASP.NET MVC

## Descripción
Aplicación web CRUD para gestión de una clínica universitaria, desarrollada con ASP.NET MVC, Entity Framework y SQL Server.

## Tecnologías
- C# / ASP.NET MVC (.NET Framework)
- Entity Framework 6
- SQL Server
- Bootstrap

## Entidades del sistema
- Pacientes
- Médicos
- Especialidades
- Citas
- Historia Clínica
- Medicamentos
- Recetas
- Facturas
